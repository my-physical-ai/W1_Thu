CHAPTER 01

Agentic AI란?
오케스트라처럼 움직이는 AI

단순히 답변하는 AI를 넘어, 스스로 생각하고 계획하고 행동하는 인공지능. AI Agent와의 차이, 추론 능력이 왜 혁신인지 살펴봅니다.

개념 비교

AI Agent vs Agentic AI

에이전트형 AI는 오케스트라 지휘자처럼 개별 AI 에이전트 전체를 지휘합니다

🤖 AI AGENT
  • 개별 과업 처리에 특화
  • 단일 LLM + 도구 사용
  • 지시받은 범위 내 행동
  • 오케스트라의 연주자
🧠 AGENTIC AI
  • 다수의 AI 에이전트를 조율·지시
  • 복수 에이전트 시스템 전체 관리
  • 복잡한 다단계 목표 자율 완료
  • 오케스트라의 지휘자
🎯

자율적 판단

최소한의 인간 개입으로 독립적으로 행동하고, 결정을 내리고, 복잡한 다단계 작업을 완료

🔄

능동적 학습

학습하고, 계획하고, 환경에 적응하여 목표를 달성하는 능동적 지능 시스템

🔗

멀티 에이전트

DB·LLM·Vector DB·Action 모듈을 연결하는 Data Flywheel과 Model Customization 구조

추론의 혁신

왜 추론 능력이 혁신적인가?

Test-Time Scaling — "더 오래 생각할수록 더 좋은 답을 낸다"

❌ 기존 생성형 AI(LLM)의 한계

  • • 입력 → 출력을 빠르게 수행하는 데만 초점
  • • 복잡한 업무·물리적 제어 처리 부족
  • • 로봇·자동차 등 연속적 판단/행동 루프 미지원
  • • 단발성(One-shot) 또는 몇 단계 Chain-of-Thought만 가능

✅ Agentic AI의 혁신 포인트

  • • 관찰→이해→계획→행동 연속 루프 자율 수행
  • • 로봇, 자율주행, 자동화 시스템에 직접 적용
  • • 추론(Reasoning) 연산량이 기존 예상을 크게 초과
  • • 새로운 아키텍처·컴퓨팅 모델 필요성 대두

💡 핵심 차별화

단순 생성에서 생각하고 행동하는 AI로의 패러다임 전환

Jensen Huang: "AI will be talking to itself" — 내부적으로 여러 추론 경로를 돌려 비교·검증

Layer 1 — AI 모델링 (Research Layer)
  • ├ 모델 설계 (Transformer, Diffusion, etc.)
  • ├ 학습 데이터, GPU, 파라미터 조정
  • └ 결과물: GPT / Gemini / Claude / LLaMA

CHAPTER 03

Physical AI
진화의 4단계

2012년 AlexNet부터 자율주행과 범용 로보틱스까지. Jensen Huang이 제시한 4단계 AI 발전 경로와 5단계 기술 전환점을 살펴봅니다.

4단계 진화

Jensen Huang이 제시한 4단계 AI 진화

Perception AI → Generative AI → Agentic AI → Physical AI

1

Perception AI (2012~)

AlexNet 출발. 음성인식, 딥 추천, 의료 영상 분석 등 인식·분류 중심

2

Generative AI

디지털 마케팅, 콘텐츠 창작. 텍스트·이미지·영상 생성. ChatGPT, DALL-E의 시대

3

Agentic AI ← 지금!

코딩 어시스턴트, 고객 서비스, 환자 케어. 다단계 작업 자율 완수

4

Physical AI → 미래!

자율주행차, 범용 로보틱스. 물리 세계와 상호작용하며 실제 환경에서 행동

5단계 전환점

Physical AI 기술 전환점

구분핵심 변화기술적 전환점
① LLM 시대언어 추론 중심, 거대 서버급 모델GPT-3/4, Gemini, Claude
② LMM 시대경량화, 엣지·온디바이스 대응Phi-3, Mistral, Gemma
③ VLA 시대Vision+Language+Action 통합, 로봇 적용ALOHA, SmolVLA, ACT
④ 월드모델 시대세상 시뮬레이션·예측 기반 계획Dreamer, WorldDreamer, RoboDreamer
⑤ 통합 루프월드모델(상상) + VLA(행동) 결합지능형 로봇 완성 구조
2018~2022

거대 LLM — GPT-3/4, Gemini, Claude. 100B↑ 파라미터, 언어추론·대화·지식, 텍스트 중심

2023~2024

경량 LMM — Phi-3, Mistral, Gemma. 임베디드·로컬 실행 가능, 멀티모달 확장 용이

2024~2025

VLA — ALOHA, SmolVLA, ACT+VLM. 비전+언어+행동 통합, 20~50Hz 실시간 제어 Loop

2025~

월드모델 — Dreamer, WorldDreamer, RoboDreamer. 세상 동역학 학습, 잠재공간 시뮬레이션, 계획(MPC)+예측

CHAPTER 04

VLA와
로봇 AI의 실현

Vision-Language-Action 모델이 로봇에게 눈과 언어와 행동을 동시에 부여합니다. 로봇 파운데이션 모델로 가는 길을 살펴봅니다.

VLA 핵심 개념

VLA: Vision · Language · Action

인터넷 규모 데이터로 학습한 VLM의 일반화 능력을 활용해 로봇의 저수준 동작(low-level actions)을 예측

👁️

VISION (시각)

카메라를 통해 세상을 봅니다. 이미지·영상·센서 데이터를 토큰으로 인코딩

💬

LANGUAGE (언어)

사람의 명령을 이해합니다. "노란 통에 물건을 집어 넣어라" 등 자연어 지시

🦾

ACTION (행동)

물리적 임무를 수행합니다. 모터 제어 신호(관절 위치·속도·EEF 포즈) 출력

+

로봇 파운데이션 모델

Robot Foundation Model &
World Foundation Model

🤖 RFM (Robot Foundation Model)

어떤 로봇이든, 어떤 작업이든 수행할 수 있도록 다양한 환경·센서·action 데이터를 대규모로 사전학습한 범용 로봇 지능 모델. VLA가 나아가야 할 방향

LLM은 SFT로 '공부'하고 RLVR로 '연습'하면서 진짜 똑똑해졌다.
VLA도 '보고 따라 하고' 점점 모든 로봇이 쓰는 큰 두뇌가 된다.

🌍 World Foundation Model

LLM처럼, 로봇·자율주행차 등 실제 물리환경과 상호작용하는 세계 기반 모델(WFM). 다음 세대 AI의 핵심

  • 🔵 Embodied Agent가 가상 시뮬레이션(rollout) 수행
  • 🟢 효율적인 action 탐색 → 강화학습(RL) 활용
  • 🟡 모방학습(IL)만으로 어려운 일반화 문제 극복
  • 🔴 상위: 사고 / 하위: 행동 이중 구조

이미지 토큰 + 텍스트 토큰 → World Foundation Model → Action Tokens → 🤖 로봇 실행

CHAPTER 05

스케일링 법칙:
하나에서 셋으로

Pre-training, Post-training, Test-time Scaling — AI 지능을 높이는 세 가지 방법을 이해하면 왜 추론형 AI가 혁신인지 알 수 있습니다.

3가지 스케일링

From One to Three Scaling Laws

컴퓨팅(Compute)을 늘릴수록 AI 지능(Intelligence)이 올라가는 세 가지 방법

1

Pre-Training Scaling

사전학습 스케일링 — "기초 체력 키우기"
모델 처음 키울 때, 데이터·파라미터·컴퓨트 확장

✅ 폭넓은 상식·언어 능력, 전천후 기초력 상승
⚠️ 비용 급증, 도메인 특화·정밀 추론 한계
2

Post-Training Scaling

사후학습 스케일링 — "코치와 스파링"
사전학습 이후 SFT, RLHF/RLAIF로 마무리 손질

✅ 말투·안전성·정확성·도메인 성능 향상
⚠️ 과적합/편향 위험, 데이터 품질 의존
3

Test-Time Scaling ✨

추론 시 스케일링 — "시험장에서 더 오래 생각하기"
가중치 변경 없이 여러 경로 탐색·자기검증

✅ 사실성↑, 복잡 문제 해결력↑, 근거 제시
⚠️ 응답 지연/비용 증가, 에이전트 오케스트레이션 필요
Test-Time 심화

추론 시 스케일링
"Reasoning"이 핵심인 이유

🔧 대표 기법

Self-Consistency — 다중 샘플→투표
CoT / ToT — 단계/트리 추론
ReAct / Reflexion — 생각↔행동 반복
RAG — 근거 검색
Tool Use — 계산기/SQL/검색/코드 실행

📊 상황별 추천 스케일링

상황추천

말투·안전·도메인 맞춤 필요Post-training
사실성·정확 근거·최신성 중요Test-time
