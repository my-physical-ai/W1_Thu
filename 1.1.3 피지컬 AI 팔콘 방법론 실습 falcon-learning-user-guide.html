◆ STEP 1

GitHub에서 강의 자료 다운로드

STEP
1
📥 GitHub에서 오늘 자료 받기
⏱️ 소요 시간 약 1분

먼저 오늘 학습할 자료가 올라간 GitHub 주소로 접속합니다. 강의실에서 강사님이 알려주신 URL을 그대로 사용하세요. 예시는 1주차 1섹션 (1-1)입니다.

🔒 https://github.com/my-physical-ai/week-one/blob/main/1-1.physical_ai_6week_curriculum_v2.html
📄 1-1.physical_ai_6week_curriculum_v2.html 2 days ago
📄 1-2.hardware_setup_v1.html 5 days ago
📄 1-3.dev_environment_v1.html 1 week ago
# 페이지 우측 상단에 있는 버튼:
⬇️ Download raw file
  1. 웹 브라우저를 열고 강사님이 알려주신 GitHub URL을 입력합니다.
  2. 해당 주차 파일(예: 1-1.physical_ai_6week_curriculum_v2.html)을 클릭합니다.
  3. 페이지 우측 상단의 "⬇️ Download raw file" 버튼을 클릭합니다.
  4. HTML 파일이 PC의 다운로드 폴더에 저장됩니다.
💡 파일명 해석

파일명 앞의 1-1은 1주차 - 1섹션이라는 뜻입니다. 2-3은 2주차 3섹션, 6-1은 6주차 1섹션입니다. 매주 그 주차의 3개 섹션을 모두 받아두면 좋습니다.

⚠️ 자주 하는 실수

GitHub 화면에서 보이는 HTML을 그대로 복사·붙여넣기 하지 마세요. 반드시 "Download raw file" 버튼으로 파일 자체를 다운로드해야 합니다. 그래야 Claude Projects에 깨끗하게 첨부됩니다.

◆ STEP 2

Claude Projects에 전용 작업공간 만들기

STEP
2
🪺 Claude에서 프로젝트 생성
⏱️ 소요 시간 약 1분

Claude.ai에 로그인한 후, "Projects" 메뉴에서 새 프로젝트를 만듭니다. 이 프로젝트가 여러분의 6주 학습 전용 작업공간이 됩니다. 대화 내용도, 첨부 파일도 모두 여기 안전하게 보관됩니다.

🔒 https://claude.ai/projects
# 좌측 사이드바
▸ Chats
▾ Projects
└ + New Project

# 프로젝트 생성 시 입력 항목:
Project Name: Falcon Learning - Physical AI
Description: 6주 Physical AI 학습 작업공간
  1. 웹 브라우저에서 claude.ai에 접속하고 로그인합니다.
  2. 좌측 사이드바에서 "Projects" 메뉴를 클릭합니다.
  3. "+ New Project" 또는 "Create Project" 버튼을 클릭합니다.
  4. 프로젝트 이름을 입력합니다 — 예: "Falcon Learning - Physical AI"
  5. 설명란에 간단히 작성합니다 — 예: "6주 Physical AI 학습 작업공간"
  6. "Create" 버튼을 눌러 완료합니다.
💡 프로젝트의 위력

일반 채팅과 달리 프로젝트는 대화 히스토리와 첨부 파일이 영구 보존됩니다. 일주일 전 했던 대화도, 오늘 첨부한 파일도, 6주차에 다시 활용할 수 있습니다.

◆ STEP 3

다운로드한 파일을 프로젝트에 첨부

STEP
3
📎 Project Knowledge에 자료 업로드
⏱️ 소요 시간 약 1분

이 단계가 Falcon Learning의 핵심입니다. 방금 다운로드한 HTML 파일을 프로젝트의 Project knowledge 영역에 업로드하면, AI가 이 자료를 모든 대화의 기준으로 삼게 됩니다. 할루시네이션이 사라지는 마법의 순간입니다.

🔒 claude.ai/project/falcon-learning
# 프로젝트 화면 우측 영역:

📚 PROJECT KNOWLEDGE
📄 1-1.physical_ai_6week_curriculum_v2.html ✓ Loaded
+ Add Content 또는 파일 드래그앤드롭

# ↑ 이 영역에 첨부된 모든 파일은
# 대화할 때마다 자동으로 참조됩니다.
  1. 방금 만든 프로젝트로 들어갑니다.
  2. 오른쪽 화면에 "Project knowledge" 영역이 보입니다.
  3. "+ Add Content" 버튼을 클릭하거나, 다운로드한 HTML 파일을 드래그앤드롭합니다.
  4. 업로드가 완료되면 파일명 옆에 "✓ Loaded" 표시가 나타납니다.
  5. 이제 이 파일이 모든 대화의 참조 기준이 되었습니다.
🎯 핵심 포인트

파일을 한 번만 첨부하면, 이후 같은 프로젝트에서 시작하는 모든 새 대화가 자동으로 이 파일을 참조합니다. 매번 다시 올릴 필요 없습니다.

📚 여러 자료 동시 첨부

주차별 자료뿐 아니라 Falcon Learning 방법론 문서, 강사가 제공한 추가 자료도 같이 첨부할 수 있습니다. 첨부 파일이 많을수록 AI의 답변이 더 정확해집니다.

🦅 Falcon Learning만의 차이

일반 ChatGPT나 Claude에 같은 질문을 하면 일반론적이거나 잘못된 답이 나옵니다. 그러나 자료가 첨부된 프로젝트에서는 — 강사가 의도한 핵심을 정확히 짚어주는 답이 나옵니다. 이것이 바로 매(Falcon)처럼 정확하게 목표를 포착한다는 의미입니다.

👤 USER · 재직자 담당자

🦅 CLAUDE · 첨부 자료 기반 응답

첨부하신 Falcon Learning 자료를 분석해서 재직자 교육 관점에서 핵심 장점 3가지를 정리해드릴게요.

1️⃣ 학습 시간이 80% 줄어듭니다
재직자는 시간이 가장 비싼 자원입니다. 일반 교육은 이론부터 처음 정독하느라 오랜 시간이 걸리지만, Falcon Learning은 강사가 수일간 압축한 핵심 자료에서 바로 시작합니다. 업무에 복귀할 수 있는 시간이 빨라집니다.

2️⃣ AI가 헛소리하지 않습니다 (할루시네이션 0)
일반 ChatGPT를 회사 업무에 쓰면 그럴듯한 거짓말이 나와 위험합니다. Falcon Learning은 강사 자료를 AI 프로젝트에 첨부하므로, AI가 그 자료를 기준으로만 답변합니다. 현장 용어 해석도 일관되어, 부서 간 커뮤니케이션 오류가 줄어듭니다.

3️⃣ 1대1 멘토가 24시간 옆에 있는 효과
재직자는 강의장에서 끝까지 묻지 못합니다. Falcon Learning은 같은 자료에 대해 무제한 반복 질문이 가능합니다. 이해 안 되면 다른 각도로 다시 묻고, 우리 회사 사례에 맞춰 응용 질문도 할 수 있습니다.

💡 한 줄 요약 — "Falcon Learning은 재직자가 일하면서도 빠르고 정확하게 배울 수 있는 유일한 방법입니다."

👤 USER

완벽해. 그러면 임원 발표용 슬라이드 3장 분량으로 다시 정리해줘. 각 슬라이드 제목 + 핵심 메시지 1줄 + 부연설명 2~3줄 형식으로.

🦅 CLAUDE

네, 임원 발표용 슬라이드 3장 분량으로 재구성하겠습니다.

📊 슬라이드 1 — 시간 효율
제목: "재직자 교육, 80% 빨라집니다"
메시지: Falcon Learning은 압축된 핵심 자료에서 즉시 출발합니다.
부연: 강사가 수일간 정제한 자료를 AI에 한 번 첨부하면, 재직자는 이론을 처음부터 정독할 필요 없이 핵심부터 학습합니다. 업무 복귀 시간이 단축됩니다.

📊 슬라이드 2 — 정확성
제목: "AI 헛소리, 0으로 만듭니다"
메시지: 강사 자료가 AI 답변의 기준이 됩니다.
부연: 일반 AI는 그럴듯한 거짓 답변(할루시네이션)이 잦아 업무에 위험합니다. Falcon Learning은 첨부 자료에서만 답을 끌어내므로, 정확성이 보장됩니다.

📊 슬라이드 3 — 24시간 멘토
제목: "강의 후에도 학습이 계속됩니다"
메시지: AI가 24시간 1:1 멘토 역할을 수행합니다.
부연: 재직자는 강의장에서 모든 질문을 하지 못합니다. Falcon Learning은 같은 자료에 대해 무제한 반복 질문이 가능하며, 우리 회사 사례에 맞춘 응용 질문도 받습니다.
